# 若木云卡管理后台

若木云卡管理后台，用于管理商品、订单、用户、支付渠道和系统设置。

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS

## What This App Does

- Dashboard and operational analytics
- Product/category/content management
- Order and fulfillment management
- Payment channel configuration
- Admin role and permission management

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
